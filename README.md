# 🎌 Anime Dashboard

An interactive Anime Analytics Dashboard built using **Python, Pandas, Plotly, and Streamlit** to explore anime data through dynamic visualizations and filters.

## 🚀 Live Demo
https://anime-dashboard-4rjk4tuvy4dfttfonur7x6.streamlit.app/

## ✨ Features
- Interactive Dashboard
- Search & Filter Anime
- Genre-wise Analysis
- Rating Distribution
- Top Rated Anime
- Interactive Charts using Plotly
- Clean & Responsive UI

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Plotly
- Streamlit
